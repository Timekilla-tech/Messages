\chapter{Системийн шинжилгээ}

\section{Системийн зорилго ба ач холбогдол}
Энэхүү төслийн үндсэн зорилго нь хэрэглэгчдийн өдөр тутмын мэдээлэл солилцоог оновчлох, мессеж харилцааг илүү зохион байгуулалттай болгох зорилгоор Fossify Messages аппликейшныг өргөжүүлэн хөгжүүлэхэд оршино~\cite{fossify_messages}. Орчин үеийн хэрэглэгчдийн өдөр тутам хүлээн авдаг банкны гүйлгээний мэдэгдэл, нэг удаагийн код (OTP) болон маркетингийн шинж чанартай их хэмжээний зурвасуудыг Regex (тогтмол илэрхийлэл) ашиглан автоматаар ангилж, хавтаслах системээр шийдсэнээр хэрэглэгчийн мэдээлэл хайх хугацааг товчилж, бүтээмжийг нэмэгдүүлэх ач холбогдолтой~\cite{android_docs}.

\section{Хэрэглэгчийн дүрүүд}
\begin{enumerate}
    \item \textbf{Ердийн хэрэглэгч (Casual User)}: Хувийн харилцаанд голчлон ашигладаг. Ирсэн зурвасуудыг цаг алдалгүй унших, хэрэгцээгүй зурвасыг хурдан устгах, энгийн бөгөөд ойлгомжтой UI/UX-ийг эрхэмлэдэг хэрэглэгч.
    \item \textbf{Мэдээллийн ачаалал ихтэй хэрэглэгч (Power User)}: Олон төрлийн үйлчилгээний мэдэгдэл, санхүүгийн мэдээлэл хүлээн авдаг. Regex болон түлхүүр үг ашиглан шүүлтүүр үүсгэж, мэдээллийн урсгалаа өөрийн хэрэгцээнд тохируулан системчлэх шаардлагатай хэрэглэгч.
    \item \textbf{Орчин үеийн төхөөрөмж эзэмшигч (Modern Device User)}: Нугалах дэлгэц (Foldable) болон таблет ашигладаг. Дэлгэцийн талбайг үр ашигтай ашиглаж, хоёр баганат (split-view) горимоор олон үйлдэл зэрэг гүйцэтгэх (multi-tasking) сонирхолтой хэрэглэгч.
\end{enumerate}

\section{Функционал шаардлагын тодорхойлолт}

\begin{enumerate}[label=FR\arabic*.]
    \item \textbf{Динамик хавтас (Folders/Tags)}: Хэрэглэгч өөрийн хэрэгцээнд тохируулан (``Орлого зарлага'', ``Нийлүүлэгч'', ``Төсөл'') нэрлэсэн хавтаснууд үүсгэх, удирдах.
    \item \textbf{Чат өнгөөр ялгах (Color-Coding)}: Тухайн харилцагч эсвэл бүлэг зурваст зориулж дэвсгэр өнгө оноох замаар харагдах байдлыг ялгах.
    \item \textbf{Хугацаагаар бүлэглэх (Age Grouping)}: Ирсэн зурвасуудыг Өнөөдөр, Өчигдөр, Энэ долоо хоног, Энэ сар гэсэн гарчиг дор автоматаар ангилах.
    \item \textbf{Regex Pattern Matching}: Тогтмол илэрхийлэл (Regular Expression) ашиглан ирж буй зурвасуудыг агуулга болон дугаараар нь таньж, зохих хавтсанд автоматаар шилжүүлэх.
    \item \textbf{Автомат ангилал}: Систем ирж буй зурвасыг хэрэглэгчийн тодорхойлсон дүрмийн дагуу шууд ангилах.
    \item \textbf{Хажуу тийш чирж удирдах (Swipe Actions)}: Зурвас дээр баруун, зүүн тийш шудрах үйлдлээр Устгах, Архивлах, Блоклох, Уншсан төлөв өөрчлөх, Онцлох үйлдлүүдийг хурдан гүйцэтгэх.
    \item \textbf{Өргөтгөсөн цэс (Long-press Menu)}: Зурвас дээр удаан дарахад гарч ирэх цэсээр өнгө солих, хавтсанд шилжүүлэх үйлдлүүдийг хийх.
    \item \textbf{Foldable Support}: Нуглардаг дэлгэц нээгдэх үед ``List-Detail'' харагдац руу автоматаар шилжих.
    \item \textbf{Default SMS App}: Android үйлдлийн системийн ``Default SMS App'' эрхийг зөв авах, системийн баазтай (SMS Content Provider) алдаагүй ажиллах.
\end{enumerate}

\section{Функционал бус шаардлага}
\begin{itemize}
    \item \textbf{Гүйцэтгэл}: 1,000+ conversation-той үед жагсаалт секундэд 60 frame (smooth scroll) хурдтай ажиллах.
    \item \textbf{Аюулгүй байдал}: Зурвасын агуулгыг гадагш дамжуулахгүй, зөвхөн локал төхөөрөмж дээр шүүлтүүр ажиллуулах.
    \item \textbf{Найдвартай байдал}: Migration үед өгөгдөл алдагдахгүй байх~\cite{room_guide}.
    \item \textbf{Кодын чанар}: Програмын эх код нь ойлгомжтой, цаашид өргөжүүлэх боломжтой (maintainable) байх зарчмыг баримталсан~\cite{clean_code}.
\end{itemize}
